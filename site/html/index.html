About the site

This website is where members of VEX robotics team 9573X can submit their changes to the competition notebook.
The notebook will then be submitted digitally. For now, much of the process is manual, but it will eventually be automated.


Technical Details

This is hosted on nginx open-source runs on a Pi 4 8gb
, and is open source on Github. If you notice bugs or something in the site, the best place
to put them so that the devs know is there. Feel free to submit pull requests as well if you want to make any improvements!
