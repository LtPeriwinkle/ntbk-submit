how to submit

PLEASE read this guide before submitting!

Since this notebook will be submitted at VEX competitions, it is VERY important that submissions be uniform.


General

It is extremely important that all changes are documented in the engineering notebook. The system relies on having every single change
documented, so that only images taken after the change are needed. All changes in a given area in a single day should be condensed into one submission.
For example, if you modify the drivetrain and someone else modifies the lift, put both of those into the same notebook submission. If this sort of
situation occurs, make sure to credit everyone who worked on it in the submission, and to include images of all of the changes.
Also, please avoid using pronouns like "I" in the notebook.



Markdown

The submission box supports a simple formatting language called Markdown. It can be used to make text bold, italicized, or even format it as code.
Here are some of the more common formatting elements available in Markdown.


What you type: What you see:
**bolded** bolded
*italicized* italicized
`monospace` 
monospace
```code``` 
code

A reference containing more advanced features of Markdown can be found here.
When using Markdown in your submission, to the right of the submission form, you will see a real-time preview of how it will look.



Images

If your submission includes modification to the robot or design, or creation of concept drawings you MUST submit images.
This is extremely important. Make a habit of taking these photos or screenshots every time you make a modification.
Ensure that the images clearly show the change, and that they are focused and have good lighting. Only submit images from
after you make the modification.



Description

On the submission page is a large text box for describing your modification. In this area, you should describe both
why you made the change and what you changed. Not to sound like an english teacher, but it is large for a reason, so use detail and be specific.



Sign your name

The last field on the submissions page is for your name. Please put your first and last name as you would sign them
in your engineering notebook for class. This is important, as we want to make sure everyone gets credit for their work. Also,
it is good practice in general to sign your name for engineering notebook entries, so get into the habit of it if you are new.
